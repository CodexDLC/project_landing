# Django Backend

## Обзор

Django-бэкенд — основной серверный компонент проекта, построенный на Django и Django REST Framework.

## Основные компоненты

- **core/** — ядро проекта: настройки, конфигурация, базовые классы
- **features/** — функциональные модули (фичи), каждый в своей директории
- **database/** — модели, миграции, схемы БД

## Принципы

- Разделение настроек по окружениям (dev / prod / test)
- Feature-based структура приложения
- Изоляция схем базы данных

## Быстрый старт

1. Установить зависимости через Poetry
2. Применить миграции: `python manage.py migrate`
3. Запустить сервер: `python manage.py runserver`
