# Ядро Django

## Обзор

Модуль `core` содержит центральную конфигурацию Django-проекта: настройки, URL-маршрутизацию и базовые утилиты.

## Split Settings

Настройки разделены на несколько файлов по окружениям:

- **base.py** — общие настройки для всех окружений
- **dev.py** — настройки для локальной разработки
- **prod.py** — настройки для продакшена
- **test.py** — настройки для тестов

## Структура

```
core/
├── settings/
│   ├── base.py
│   ├── dev.py
│   ├── prod.py
│   └── test.py
├── urls.py
└── wsgi.py
```

Выбор окружения задаётся через переменную `DJANGO_SETTINGS_MODULE`.
