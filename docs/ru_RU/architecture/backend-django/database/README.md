# База данных

## Обзор

Работа с базой данных построена на Django ORM с поддержкой изоляции схем и управляемых миграций.

## Ключевые аспекты

- **Django ORM** — модели описываются как Python-классы
- **Миграции** — автоматическое версионирование структуры БД
- **Изоляция схем** — разделение данных по PostgreSQL-схемам

## Команды

```bash
python manage.py makemigrations   # создать миграции
python manage.py migrate          # применить миграции
```

## Принципы

- Каждая фича хранит свои модели и миграции
- Схемы БД изолированы для мультитенантности
- Миграции всегда под контролем версий
